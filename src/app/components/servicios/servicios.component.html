diseño gráfico - producciones fotograficas

Realizamos una planificación de contenidos para redes sociales, web o gráficas. Este puede tener diseño gráfico o intervenciones gráficas o bien producciones fotográficas.

producciones fotograficas

Se puede contratar con un paquete de redces o se pueden hacer por una sola producción. Incluye:
-Idea creativa
Negociación y coordinación con el equipo
-Entrega de material final.
El servicio no incluye el valor del equipo como fotógrafos, modelos, maquilladores o locacion.
